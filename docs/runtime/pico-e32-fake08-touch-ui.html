On-screen touch controls

FT6236 capacitive touch behind the merged input_poll() seam — a portrait handheld: the PICO-8 screen up top, a soft control deck below.

TO SCALE · 320 × 480 PORTRAIT

Screen up top

256×256, flush to the top (drawFrame OY 112→0) so the deck is one clean band, not split margins.

Unified d-pad

One cross, four directional zones + a dead hub — bottom-left, thumb-reachable.

O / X buttons

Spherical, thin-ringed, on a gamepad diagonal, bottom-right.

2-finger play

The FT6236 reports two points, so a direction + O/X register together (move + jump). Each point → zone → a bit in input_poll().

Y-flip

Glass is mounted upside-down — raw touch-Y is flipped to match the frame; first bring-up step is a tap-a-known-spot calibration.
